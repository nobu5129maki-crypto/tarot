Lumina Tarot - 78 Origins

神秘的なデザインとAI（Gemini API）による本格的なタロット占い体験を提供するシングルページWebアプリケーションです。

🌟 プロジェクトの概要

Lumina Tarotは、直感的なインターフェースと最新のAI技術を融合させたデジタル・タロット・プラットフォームです。ウェイト版タロット全78枚を網羅し、ユーザーの引いたカードに合わせてパーソナライズされたリーディングを生成します。

🚀 機能

フルデッキ・シミュレーション: 大アルカナ22枚、小アルカナ56枚を完全実装。

AIリーディング: Gemini 2.5 Flash APIを使用した高度な解釈。

PWA対応: スマートフォンのホーム画面に追加してアプリとして利用可能。

レスポンシブ・デザイン: PC、タブレット、スマートフォン全てのデバイスに最適化。

🛠 構築方法とデプロイ

1. 構成

GitHubリポジトリを以下の構造で作成してください。

.
├── index.html      # フロントエンド本体
└── api/
    └── fortune.js  # Vercel Serverless Function (AI中継用)


2. 環境変数の設定

Vercel等のホスティングサービスの管理画面にて、以下の環境変数を設定してください。

GEMINI_API_KEY: Google AI Studioから取得したAPIキー

3. デプロイ

リポジトリをVercelに接続すると、自動的にServerless Functionが認識され、安全なAPI通信が可能になります。
